# Statistics Coursework – Mental Health Analysis

This repository contains a university statistics project focused on analyzing mental health and lifestyle patterns using survey-based data.

## Project Overview
The project analyzes how different lifestyle factors such as sleep duration and drink preferences (tea/coffee) relate to mental health conditions among people from various professions.

The dataset was collected from multiple online sources and processed for statistical analysis and visualization.

## Files Included
- Group6_Mental_Health_Analysis.csv – Cleaned dataset used for analysis
- Group6_Mental_Health_Analysis_report.pdf – Project report with explanations and visualizations

## Dataset Description
The dataset contains the following attributes:
- Text: Participant comments
- Person Type: Profession (student, employee, etc.)
- Sleep Amount: Daily sleep duration
- Drink Type: Beverage preference (tea/coffee/others)
- Mental Health: Mental health state
- Source: Data source

## Analysis Performed
- Descriptive statistics
- Exploratory Data Analysis (EDA)
- Bar charts, histograms, and pie charts
- Interpretation of mental health trends across professions

## Purpose
This project was completed as part of the Engineering Statistics course for academic learning and practice purposes.
